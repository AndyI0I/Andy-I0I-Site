What is this Website?



This is mainly just a portfolio page.
It also has the benefit that it is needed for my performance evaluation and is therefore exam relevant.
This page was made with Bootstrab v 5.0, HTML 5 and CSS 3.
Most of it should be obvious for full-stack developers, but not for some, so I'll explain a bit.




What is HTML 5?



The main difference between the older versions of HTML is that the old versions did not have the tags "main, footer, nav" etc.
which makes it difficult for screen readers and bots to understand the page.
Since the "main" is usually the most important part, like this article with its header, even a bot knows that the most
important info is given there and that it can ignore the nav, since it is is only a navigation bar for the user.




CSS... what?



CSS, oder auch Cascading Style Sheet genannt, wird benutzt um Stylesheets zu erstellen, welche nicht nur durch HTML benutzt werden können.
You can also use it with Java with "Swing", but that doesn't matter now, what is important is that you can include a css file in an HTML document
with a "link"-tag.

"Classes" and "IDs". They are used either to address the tags in Javascript or to style them in CSS. Since IDs are usually called "unique", you can use them only once per document, while you can use classes can be used indefinitely. In CSS classes are addressed with a "." in front and IDs with a "#", IDs are also case sensitive.




What else is there in CSS?



In CSS you can not only give divs a nice border or change the style of a paragraph, if you want and have the time you can also add animations and transitions. While transitions are used for transitions, so that changes don't look so stiff, animations are used to bring a little life into a website, e.g. you make a pulsating word!
