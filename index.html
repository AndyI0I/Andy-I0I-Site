Was ist HTML 5?



Der Hauptsächliche Unterschied zwischen den älteren Versionen von HTML ist, dass die alten versionen nicht die tags "Main, footer, nav" usw. hatten
was es schwierig für screen reader und bots macht, die seite zu verstehen.
Da meist im "Main" das wichtigste ist, wie diese article mit deren header, weiss auch ein bot das dort die Wichtigsten infos gegeben sind und das er das Nav ignorieren kann, da es
nur eine navigier leiste ist für den Nutzer.




CSS... was?



CSS, oder auch Cascading Style Sheet genannt, wird benutzt um Stylesheets zu erstellen, welche nicht nur durch HTML benutzt werden können.
Man kann es auch mit Java benutzen mit "Swing", aber das ist jetzt egal, was wichtig ist, ist dass man ein css File in einem HTML Dokument
einbinden kann per "link"-tag.

Wie der Name es eigentlich erklärt, werden die einzelnen veränderungen "Kaskadierend" prioisiert, dass heisst, das letzte was geschrieben wird, wird immer am meisten Priotisiert, aber dass ist erst relevant wenn man mehrmals den gleichen Tag anspricht mit mehreren "classes" und "IDs"

Apropo "Classes" und "IDs". Sie werden benutzt um entweder in Javascript die Tags anzusprechen oder in CSS sie zu stylen. Da man meist IDs als "einzigartig" bezeichnet, kann man sie auch nur einmal pro dokument benutzen, währenddessen man classes unendlich benutzen kann. In CSS werden Classes mit einem "." vorne angesprochen und IDs mit einem "#", IDs sind auch sensible auf Gross und kleinschreibung.




Was gibt es noch in CSS?



In CSS kann man nicht nur "divs" einen schönen Rand geben oder die Schrifft-art von einem Paragraph ändern, wenn man will und die Zeit dafür hat, kann man auch Animationen hinzufügen und "transitions". Während Transitions für übergänge benutzt werden, damit veränderungen nicht so steif wirken, wernden Animationen benutzt um ein bisschen leben in eine Website zu rufen, z.b. man macht ein Pulsierendes Wort!
